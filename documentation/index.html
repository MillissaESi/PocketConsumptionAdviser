Welcome to Pocket Consumption Adviser! This is our API documentation

Pocket Consumption Adviser

This is our high-quality API. We use this API to request the server to get the best recommendation to reduce the food waste

Get the recommendation for a given product

Get the recommendation for the current user for a given product.

Property Description
product (required) the id of the product
origin (required) (from_scanner, from_new_list, from_online_purchase)
GET
/pca/v1/recommendations/{product}/{origin}

Example request

$ curl -H "Authorization: OAuth <ACCESS_TOKEN>" https://pocket-consumption-adviser.com/pcas/v1/recommendations/{product}/{origin}

Example response

{
    "owner": "{user}",
    "id": "{recommendation_id}",
    "product": "{product}",
    "quantity": "{quantity}",
    "created": "{timestamp}"
}

Get a question about a given product

Get a question for the the current user about a given product.

Property Description
product (required) the id of the product
origin (required) (from_scanner, from_new_list, from_online_purchase)
GET
/pca/v1/questions/{product}/{origin}

Example request

$ curl -H "Authorization: OAuth <ACCESS_TOKEN>" https://pocket-consumption-adviser.com/pcas/v1/questions/{product}/{origin}

Example response

{
  "owner": "{user}",
  "id": "{question_id}",
  "body": "{question_body}",
  "product": "{product}",
  "quantity": "{quantity}",
  "created": "{timestamp}"
}

Answer a question

Answer a given question.

POST
/pcas/v1/questions/{question}/answer

Example request

curl -X POST https://pocket-consumption-adviser.com/pcas/v1/{question}/answer \
  -X PUT \
  -d @file.json

Example request body

{
  "answer": "yes"
}

Example response

{
  "owner": "{user}",
  "question": "{question_id}",
  "answer": "yes",
  "description": null,
  "created": "{timestamp}"
}

Retrieve a pca

Returns a single pca.

Retrieve information about an existing pca.

GET
/pcas/v1/{username}/{pca_id}

Example request

curl https://pocket-consumption-adviser.com/pcas/v1/{username}/{pca_id}

Example response

{
  "owner": "{username}",
  "id": "{pca_id}",
  "created": "{timestamp}",
  "modified": "{timestamp}"
}

Update a pca

Updates the properties of a particular pca.

Property Description
name (optional) the name of the pca
description (optional) a description of the pca
PATCH
/pcas/v1/{username}/{pca_id}

Example request

curl --request PATCH https://pocket-consumption-adviser.com/pcas/v1/{username}/{pca_id} \
  -d @data.json

Example request body

{
  "name": "foo",
  "description": "bar"
}

Example response

{
  "owner": "{username}",
  "id": "{pca_id}",
  "name": "foo",
  "description": "bar",
  "created": "{timestamp}",
  "modified": "{timestamp}"
}

Delete a pca

Deletes a pca, including all wibbles it contains.

DELETE
/pcas/v1/{username}/{pca_id}

Example request

curl -X DELETE https://pocket-consumption-adviser.com/pcas/v1/{username}/{pca_id}

Example response

HTTP 204

List wibbles

List all the wibbles in a pca. The response body will be a pcaCollection.

GET
/pcas/v1/{username}/{pca_id}/wibbles

Example request

curl https://pocket-consumption-adviser.com/pcas/v1/{username}/{pca_id}/wibbles

Example response

{
  "type": "pca",
  "wibbles": [
    {
      "id": "{wibble_id}",
      "type": "pca",
      "properties": {
        "prop0": "value0"
      }
    },
    {
      "id": "{wibble_id}",
      "type": "pca",
      "properties": {
        "prop0": "value0"
      }
    }
  ]
}

Insert or update a wibble

Inserts or updates a wibble in a pca. If there's already a wibble with the given ID in the pca, it will be replaced. If there isn't a wibble with that ID, a new wibble is created.

Property Description
id the id of an existing wibble in the pca
PUT
/pcas/v1/{username}/{pca_id}/wibbles/{wibble_id}

Example request

curl https://pocket-consumption-adviser.com/pcas/v1/{username}/{pca_id}/wibbles/{wibble_id} \
  -X PUT \
  -d @file.geojson

Example request body

{
  "id": "{wibble_id}",
  "type": "pca",
  "properties": {
    "prop0": "value0"
  }
}

Example response

{
  "id": "{wibble_id}",
  "type": "pca",
  "properties": {
    "prop0": "value0"
  }
}
